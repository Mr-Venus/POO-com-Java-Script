# 📘 POO com Java Script.

### Este repositório foi criado com o objetivo de **armazenar meus projetos e exercícios de POO desenvolvidos em JavaScript**, servindo como um espaço de **aprendizado, prática e evolução contínua** ao longo do semestre.

## 🎯 Objetivo

O principal objetivo deste repositório é:
- Praticar os conceitos fundamentais de **POO em JavaScript**
- Registrar minha **evolução técnica** durante a disciplina
- Consolidar o aprendizado por meio de projetos práticos
- Criar um histórico de estudos para consultas futuras

## 📂 Organização

Cada pasta do repositório representa um projeto, exercício ou atividade desenvolvida durante os estudos, podendo conter:
- Código-fonte
- Comentários explicativos
- Ajustes e melhorias feitas com o tempo
